# Seller AI Assistant — Личный кабинет продавца

Веб-приложение для управления объявлениями с интегрированным AI-ассистентом. Помогает продавцам оптимизировать описания товаров и оценивать их рыночную стоимость с помощью LLM (Llama 3 via Groq).

## 🚀 Основные возможности
- **AI-копирайтер**: Улучшение описания товара с учетом категории и характеристик.
- **AI-оценщик**: Анализ рыночной стоимости на основе контекста объявления.
- **Умные черновики**: Автоматическое сохранение изменений в `localStorage` через **Zustand**.
- **Динамические формы**: Поля характеристик адаптируются под выбранную категорию (Авто, Недвижимость, Электроника).
- **Валидация**: Индикация объявлений, требующих доработки (на основе пустых обязательных полей).

## 🛠 Технологический стек
- **Frontend**: React 18, TypeScript, Vite.
- **UI-библиотека**: Ant Design (Antd).
- **State Management**: Zustand (Client state & Persist), React Query (Server state).
- **Network**: Axios.
- **AI Integration**: Groq Cloud API (Model: Llama 3.3 70B).
